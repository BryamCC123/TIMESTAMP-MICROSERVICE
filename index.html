Timestamp Microservice

Convert dates to UNIX timestamps and UTC strings with this simple API service

API Usage

Send a GET request to:

GET /api/:date?

Where :date? is either:

  • A valid date string (e.g., "2015-12-25")
  • A UNIX timestamp (e.g., 1451001600000)
  • Empty - returns current time

Example Response

For a valid date:

{
  "unix": 1451001600000,
  "utc": "Fri, 25 Dec 2015 00:00:00 GMT"
}

For an invalid date:

{ "error": "Invalid Date" }
